Tcp长连接，远程控制

  • 首先参考乐联网反向控制教程，新建一个控制器，这里以开关为例。
  • 原理是与服务器保持一个TCP长连接，不到40s刷新一次，以此保持通道，与被控制段通信，发送控制信息。

  • Tcp长连接参考了@墨迹流韶的Android基于Tcp协议的Socket长链接封装

  • 地址 tcp.lewei50.com
    端口号 9960
    心跳包间隔 1min以内

  • 发送/接收数据格式 Json
    本地发送数据格式

    服务器发送控制命令格式，数据处理时需要去掉字符串最后的&^!

  • 处理数据 在service的onRecived中
